Hexagonal Hole patterns¶

Hexagonal hole patterns are one way to have some ventilation for housings maded with Boxes.py. Right now both .rectangularWall() and .roundedPlate() do supports this pattern directly by passing the parameters to the calls. For other use cases these more low level methods can be used.

For now this is the only supported pattern for ventilation slots. More may be added in the future.

There is a global Boxes.hexHolesSettings object that is used if no settings are passed. It currently is just a tuple of (r, dist, style) defualting to (5, 3, ‘circle’) but might be replace by a Settings instance in the future.
